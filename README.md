# URLShortener
Simple url shortener as a serivce with go

## How use this service?
For get the short form of your url you can send a HTTP ``post`` request to [hallows.ir/add](http://halows.ir/add):
### Request
```JSON
{
  "long_url": "<Your URL>"
}
```

### Response 
```JSON
{
  "short_url": "<Shorted URL>"
}
```
